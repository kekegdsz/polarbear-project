# 开放 80 端口 - 解决无法访问

你的网站**在服务器内部已正常工作**（本地测试返回 200），无法从外网访问是因为 **云厂商安全组** 没有放行 80 端口。

## 操作步骤

根据你的服务器所在云平台，在**云控制台**中操作：

---

### 阿里云
1. 登录 [阿里云控制台](https://ecs.console.aliyun.com)
2. 左侧 **实例与镜像** → **实例**
3. 找到服务器 `183.56.251.215`，点击实例 ID
4. 点击 **安全组** → 进入该实例绑定的安全组
5. **入方向** → **手动添加**：
   - 端口范围：`80/80`
   - 授权对象：`0.0.0.0/0`
   - 协议：TCP
6. 保存

---

### 腾讯云
1. 登录 [腾讯云控制台](https://console.cloud.tencent.com/cvm)
2. 实例列表 → 找到该服务器 → 点击 **更多** → **安全组** → **配置安全组**
3. 入站规则 → **添加规则**：
   - 端口：`80`
   - 来源：`0.0.0.0/0`
   - 协议：TCP
4. 保存

---

### 华为云
1. 登录 [华为云控制台](https://console.huaweicloud.com/ecm)
2. 弹性云服务器 → 找到实例 → 安全组
3. 入方向规则 → 添加规则：
   - 端口：80
   - 源地址：0.0.0.0/0

---

### 其他云（如天翼云等）
在云控制台找到 **安全组 / 防火墙 / 入站规则** 等入口，添加入站规则：
- **端口**：80
- **协议**：TCP
- **来源**：0.0.0.0/0（允许所有 IP）

---

## 宝塔面板内再确认

放行后，到宝塔面板：
1. **安全** → 确认 80 端口已放行
2. 若无，点 **放行端口**，填入 `80`，协议选 TCP

---

完成后用浏览器访问：**http://183.56.251.215/**
